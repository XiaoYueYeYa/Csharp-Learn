## var隐式类型
var 是一种特殊的变量类型
它可以用来表示任意类型的变量
**注意**
1.var**不能作为类的成员 只能用于临时变量声明时**
也就是 一般写在函数语句块中
2.**var必须初始化**

```C#
var i = 0;
var s = "123";
var array = new int[] { 1, 2, 3, 4 };
var list = new List<int>();
```

## 设置对象初始值
```C#
class Person
{
    private int money;
    public bool sex;

    public string Name
    {
        get => "Xiao";
        set => sex = true;
    }
    public int Age
    {
        get;
        set;
    }
    public Person(int money)
    {
        this.money = money;
    }

    //知识点八
    public int Add(int x,int y)=>x+y;
    public void Speak(string str)=> Console.WriteLine(str);
}
```

```C#
//声明对象时
//可以通过直接写大括号的形式初始化公共成员变量和属性
//(默认无参构造时可省略括号)
//Person p = new Person{sex = true, Age = 18,Name="Xiao"};
Person p = new Person(100) {sex = true, Age = 18,Name="Xiao"};
Person p2 = new Person(200) {Age = 18};
```

## 匿名类型
var 变量可以申明为自定义的匿名类型
```C#
var v = new { age = 10, money = 11, name = "Xiao" };
Console.WriteLine(v.age);
Console.WriteLine(v.name);
```

## 可空类型

1.值类型是不能为 空的
```C#
int i = null;//报错
```
2.声明时 在值类型后面加? 可以赋值为空
```C#
int? c = null;
```
3.判断是否为空
```C#
if (c == null)
{
    Console.WriteLine(c);
    Console.WriteLine("空");
}
if (c.HasValue)
{
    Console.WriteLine(c.Value);
    Console.WriteLine(c);
}
```
4.安全获取可空类型值
```C#
int? value = null;
```
4-1.如果为空 默认返回值类型的默认值
```C#
Console.WriteLine(value.GetValueOrDefault());
```
4_2.也可以指定一个默认值
```C#
Console.WriteLine(value.GetValueOrDefault(100));

float? f = null;
double? d = null;
decimal? de = null;

object? o = null;
if (o != null)
{
    Console.WriteLine(o.ToString());
}
```
相当于一种语法糖 能够帮助我们自动判断o是否为空
如果是null就不会执行也不会报错
```C#
Console.WriteLine(o?.ToString());

int[] arrayInt = null;
Console.WriteLine(arrayInt?[0]);

Action action = null;
if (action != null)
{
    action();
}
```
简化
```C#
action?.Invoke();
```

## 空合并操作符
空合并操作符 ??
左边值 ?? 右边值
如果左边值为null 就返回右边值 否则返回左边
**只要是可以为null的类型都能使用**

```C#
int? intV =null;
```
三目运算符
```C#
int intI = intV == null ? intV.Value : 100;//报错
int intI = intV ?? 100;
Console.WriteLine(intI);
```

```C#
string str = null;
str = str ?? "右边值";
Console.WriteLine(str);
```

## 内插字符串
关键符号:$
用$来构造字符串 让脂肪层中可以拼接变量
```C#
string name = "Xiao";
int age = 18;
Console.WriteLine($"好好学习,{name},年龄:{age}");
```

```C#

```
